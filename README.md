# 🚀 Algorithm & Data Analysis Portfolio

本リポジトリは、Python, Java, C++ を用いたアルゴリズムの実装と、Pandas を活用したデータ分析スキルの習得記録です。

### 🛠 Tech Stack
- **Languages:** Python 3, Java 17+, C++ (C++17/20)
- **Data Analysis:** Pandas
- **Environment:** GitHub Codespaces

---

各アルゴリズムに対し、言語特性（Python の簡潔さ vs Java の型安全性 vs C++ のメモリ管理）を考慮した多言語実装を行っています。

| No. | 課題名 | 核心技術ポイント | 実装言語 |
| :--- | :--- | :--- | :--- |

### 📊 2. Pandas Data Analysis (15 Projects)
Pandas を用いたデータ前処理から分析パイプライン構築までの実践演習です。

- **データクレンジング**: 欠損値処理, 型変換, 重複排除.
- **データ操作**: フィルタリング, カラム編集, ソート.
- **高度な分析**: 多重結合 (Merge/Concat), 構造変形 (Pivot/Melt).

---

### 💡 技術的アピールポイント

* **多言語エンジニアリング**: 
    * Python（プロトタイピング）、Java（オブジェクト指向）、C++（メモリ最適化）を適材適所で使い分ける力を養っています。
* **効率的アルゴリズム**: 
    * 時間計算量 $O(N)$ だけでなく、空間計算量 $O(1)$ を目指したインプレース操作を積極的に採用。特に C++ 実装でメモリ効率を追求。
* **データドリブン志向**: 
    * Pandas を用いた一連のプロジェクトにより、データ解析のライフサイクルを全般的に遂行可能。
